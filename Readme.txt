To build and run this project you need to have .Net Framework 3.5 installed and MS Build.
This build runs against the Asp.Net MVC 1.0 and 2.0. To open the solution you need Microsoft Visual Studio 2008.
Before starting your work you should run 'Click to build.bat' to make sure everything is working as expected on your system.
